# Contributors

XR Foundry recognizes code and non-code contributions through public evidence. The
policy is defined in
[`docs/contributing/recognition-policy.md`](docs/contributing/recognition-policy.md).

This is a thank-you page, not a ranking, score table, maintainer list, CODEOWNERS
file, or repository permission record. Entries are ordered alphabetically by GitHub
identity and link to accepted contribution evidence.

## Contribution types

`code` · `docs` · `research` · `review` · `test` · `userTesting` · `infra` ·
`design` · `triage` · `translation` · `community`

## Credited contributors

No V1 contribution-credit records have been accepted yet. The first entry will be
added only with a consented, evidence-backed record validated against
[`contribution-credit.schema.json`](docs/contributing/contribution-credit.schema.json).

AI or advanced-tool assistance is disclosed in the related pull request, commit,
and evidence record. The accountable GitHub contributor remains responsible; an
assistance tool is not listed here as a human contributor or repository authority.
