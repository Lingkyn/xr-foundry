# Inventory UI Toolkit State Gallery

After importing the sample, run **GameObject > XR Foundry > Inventory > Create UI
Toolkit State Gallery**. The command creates a consumer-owned screen-space
`PanelSettings` asset and a `UIDocument` using the package's UXML. It does not
modify an existing scene or input system during package import.

Use `InventoryUIToolkitStateGalleryBootstrap.ReplayState` from the Inspector,
another script, or tests to replay empty, partial, full, rejected, selected,
disabled, loading, and error view states.

This renderer sample is not an XR or device test.
